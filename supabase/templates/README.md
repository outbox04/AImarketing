# Supabase Data Templates

This folder contains CSV templates for easy data entry without code.

## How to use
1. Open Supabase Studio.
2. Go to the table you want to fill.
3. Choose `Import` or `Upload CSV`.
4. Select the matching CSV template from this folder.
5. Review the field headers and import data.

If your team prefers manual entry, they can also open the table in Supabase Studio and add rows directly.
